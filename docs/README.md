# Audrix

Real-time speech-to-text for Windows. Captures microphone or system audio, transcribes instantly, and keeps everything local.

## What it does

Audrix listens to what's playing on your PC and turns it into text in real time. Transcribe your own voice, a YouTube video, a Zoom call, or both at once. No recording step, no waiting, no cloud upload.

- **Mic + System Audio** — Capture your microphone, any app's audio output, or both simultaneously with automatic speaker separation.
- **Noise Suppression** — Configurable noise gate and auto-gain keep background hum and silence out of your transcripts.
- **Crosstalk Detection** — Dual-stream mode distinguishes the local user from remote speakers on calls.
- **Multiple Backends** — Plug in Whisper, Lemonade NPU acceleration (AMD Ryzen), or any external transcription endpoint. Switch at runtime.
- **100% Offline** — All processing happens on your machine. Your audio never leaves.

## Getting started

Download the latest release from GitHub and run `Audrix_Community.exe`.

No installation, no dependencies, no account required.

## Requirements

- Windows 10 / 11 (64-bit)
- 4 GB RAM (8 GB recommended for larger models)
- Optional: AMD Ryzen 7040+ with XDNA NPU for hardware-accelerated transcription

## Community edition

A free community build is available. Session duration is capped at 30 minutes with cooldown periods between uses. Studio and Enterprise editions are available for extended use.
